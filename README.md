# 🍔 RedBurguer - Sistema de Pedidos

Este é um sistema de pedidos de lanches desenvolvido como um cardápio interativo para uma hamburgueria fictícia chamada **RedBurguer**.

## 🌐 Acesse o Projeto

👉 https://seu-usuario.github.io/seu-repositorio/

## 🚀 Funcionalidades

- Visualização dos lanches disponíveis  
- Adição de itens ao carrinho  
- Personalização dos lanches (remoção de ingredientes)  
- Simulação de envio do pedido para a cozinha  
- Interface interativa e intuitiva  

## 🛠️ Tecnologias Utilizadas

- HTML  
- CSS  
- JavaScript  

## 🎯 Objetivo do Projeto

O objetivo deste projeto foi praticar o desenvolvimento front-end, focando na criação de interfaces interativas e na experiência do usuário, simulando um sistema real de pedidos.
